# MenuMorph

MenuMorph is an AI-powered web application that generates a responsive restaurant website from a menu image.

## Features

- Upload restaurant menu
- AI extracts restaurant information
- AI designs website
- AI generates HTML/CSS/JS
- Responsive website output

## Tech Stack

- Python
- FastAPI
- Gemini API
- Pydantic

## Run

```bash
pip install -r requirements.txt

uvicorn app.main:app --reload
